# TP NoSQL - Projet Reseau Sociaux

Application de reseau social simple avec Streamlit et MongoDB.

## Prerequis

- Python 3.10+
- MongoDB local sur `mongodb://localhost:27017`

## Installation

```bash
pip install -r requirements.txt
```

## Lancer l'application

```bash
streamlit run app.py
```

## Structure

- `app.py` : interface Streamlit et logique principale
- `gestion.py` : fonctions complementaires du projet
